# Documentation assets

This folder contains release-facing documentation assets for `tui-alchemy`.

- `screenshots/` — curated screenshots referenced by the main README and release notes.
- `install.md` — installer, crates.io, and binary archive guidance.
- `release-v0.1.0.md` and `release-v0.2.0.md` — GitHub release note drafts.

The source of truth for player instructions is the root [`README.md`](../README.md).
